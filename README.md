# Artizana – Moroccan Handcrafted E-commerce Website

**Artizana** is a multi-page handcrafted e-commerce website dedicated to **Moroccan artisanal products**.  
It was created using **HTML, CSS, and JavaScript** to promote Morocco’s traditional craftsmanship through a clean and modern design.

---

## General Description

The **Artizana** website serves as a showcase platform for Moroccan handmade products.  
Its goal is to highlight the beauty of artisanal work while serving as a learning project in front-end web development.

The site includes several main sections and pages:

- **Home Page** – introduces the brand and features selected products.  
- **Category Pages** – each category displays a collection of handmade items with **images, product names, descriptions, and prices**.  
- **Client Feedback Section** – shows testimonials from satisfied customers.  
- **Responsive Design** – ensures compatibility with both desktop and mobile devices.

---

## Purpose

The main purpose of Artizana is to:
- Promote Moroccan artisanal heritage through a web interface.  
- Demonstrate a complete multi-page website structure.  
- Serve as a practical project for learning **HTML, CSS, and JavaScript**.  

---

## Technologies Used

- **HTML5** – page structure and content  
- **CSS3** – styling and responsive layout  
- **JavaScript (ES6)** – interactivity and dynamic components  

---

## Features

- Multi-page layout (Home + 4 categories)  
- Product cards with images and details  
- Customer testimonials section  
- Responsive design for all devices  
- Simple and elegant visual style
